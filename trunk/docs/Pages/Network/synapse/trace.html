Trace

The trace synapse is a special type of synapse designed specifically to interact with the trace neuron. It is based on Sutton and Barto's 1981, "Towards a Modern Theory of Adaptive Networks: Expectation and Prediction," which prefigures a broad body of contemporary reseaerch into reinforcement learning, and is to some extent superceded by "temporal difference" models.

Note: This synapse only works if both neurons it connects are trace neurons. Otherwise the synapse does nothing; it behaves like a clamped synapse.

See Ralph Sutton and Andrew Barto (1981), "Towards a Modern Theory of Adaptive Networks: Expectation and Prediction," Psychological Review.
