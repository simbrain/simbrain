Workspace Example

Open an existing workspace using File > Open workspace, move and resize the windows slightly, save using File > Save workspace, and reopen to observe the changes.

Select File > Clear workspace to remove all current components from the workspace, create a single network using Insert > New network, open a sample network, and open a gauge using the gauge button in the network frame. Run the network using its play button and observe the pattern which results in the gauge.
